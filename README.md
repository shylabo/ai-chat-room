# ai-chat-room

## local

1. Create an .env file server directory and set your openai api key

```
OPENAI_API_KEY={YOUR_API_KEY}
```

2. Run each node

```sh
# run client side
$ cd client/
$ npm install
$ npm run dev

# run server side, opening a new terminal
$ cd server/
$ npm install
$ npm run server
```
